Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
В программе произойдёт дедлок
Канал ch должен быть закрыт, но из за того что этого не происходит,
цикл читающий канал будет вечно ожидать новых значений
```

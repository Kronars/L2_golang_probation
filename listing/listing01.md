Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Программа выведет [77 78 79]
Создаётся массив a длинны 5 и ссылка(срез) b 
Участок с 1го элемента по 3ий, 1ый включительно, 3 - нет  

```
